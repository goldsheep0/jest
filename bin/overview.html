Ce programme, réalisé dans le cadre du projet de LO02, permet de lancer une partie du jeu de Jest.
Ce jeu est un jeu de carte à 3 ou 4 joueurs. Le but est de construire un "jest", tas de carte prise dans les offres des joueurs, contenant le plus de points.
La documentation suivante vous permettra de comprendre et de vous approprier les classes du projet.
La classe Partie est le point d'entrée de la partie.
